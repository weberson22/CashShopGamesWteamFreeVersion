# Português

## Guia do Cash Shop Editor

Este guia apresenta os principais recursos do editor, a estrutura geral do projeto, o fluxo de uso e as novidades adicionadas para melhorar a compatibilidade com diferentes bases de servidor.

### Principais Recursos

- Sem entradas inúteis nos dados do Cash Shop.
- Dados da loja centralizados, evitando registros ausentes.
- Interface amigável.
- Visual limpo e profissional.
- Tratamento de erros.
- Recursos de preenchimento automático e configuração automática.

### Estrutura de Funcionalidades

- Interface, menu, opções, configurações e organização geral do editor.
- Iniciar um novo projeto de loja.
  - Começar com categorias padrão ou categorias vazias.
- Gerenciamento fácil de categorias.
  - Adicionar nova aba principal ou categoria.
  - Renomear aba principal ou categoria.
  - Excluir aba principal ou categoria.
  - Ordenar ou reorganizar abas principais e categorias.
- Adicionar pacotes ao projeto da loja.
  - Pacote simples (1 produto dentro do pacote).
  - Pacote múltiplo (até 6 opções de produtos dentro do pacote).
  - Pacote bundle (até 6 produtos comprados em um único pacote).
    - Adicionar produto ao bundle.
    - Adicionar novo produto.
- Salvar seu projeto de loja.
- Abrir um projeto salvo.
- Editar ou excluir pacotes da loja.
  - Editar ou excluir pacote de produto único.
  - Editar ou excluir pacote de múltiplos produtos.
  - Editar ou excluir pacote bundle.
    - Editar ou excluir o bundle.
    - Editar ou excluir produto.
- Visualizar suas DataTables.
- Exportar seu projeto da loja.

### Novidades

- Adicionado suporte a servidores baseados em MueMU, Louis Emulator, Matrix Emulator e similares.
- Adicionado suporte nativo para itens Socket.
- Adicionado suporte a itens cujo level altera nome e identificação visual, como Box of Kundun.
- Adicionado suporte para enviar itens como presente diretamente pela vitrine do Cash Shop de um jogador para outro jogador.

# English

## Cash Shop Editor Guide

This guide introduces the main editor features, the overall project structure, the common workflow, and the latest improvements added to support different server bases.

### Main Features

- No useless entries in Cash Shop data.
- Centralized shop data with no missing entries.
- User-friendly interface.
- Clean and professional UI.
- Error handling.
- AutoFill and AutoSet features.

### Feature Structure

- UI, menu, options, settings, and general editor organization.
- Start a new shop project.
  - Start with default or empty categories.
- Easy category management.
  - Add new main tab or category tab.
  - Rename main tab or category tab.
  - Delete main tab or category tab.
  - Sort or arrange main tabs and category tabs.
- Add packages to the shop project.
  - Single package (1 product inside the package).
  - Multi package (up to 6 product choices inside the package).
  - Bundle package (up to 6 products in a single purchase package).
    - Add product to bundle package.
    - Add new product.
- Save your shop project.
- Open your saved shop project.
- Edit or delete your shop packages.
  - Edit or delete single product packages.
  - Edit or delete multi product packages.
  - Edit or delete bundle packages.
    - Edit or delete the bundle package.
    - Edit or delete a product.
- Preview your DataTables.
- Export your shop project.

### What's New

- Added support for server bases such as MueMU, Louis Emulator, Matrix Emulator, and similar projects.
- Added native support for Socket items.
- Added support for items whose level changes the item name and visual identification, such as Box of Kundun.
- Added support for sending items as gifts directly from the Cash Shop showcase from one player to another.

## Guía del Cash Shop Editor

Esta guía presenta las funciones principales del editor, la estructura general del proyecto, el flujo de uso y las últimas mejoras agregadas para soportar diferentes bases de servidor.

### Funciones Principales

- Sin entradas inútiles en los datos del Cash Shop.
- Datos de la tienda centralizados y sin registros faltantes.
- Interfaz fácil de usar.
- Interfaz limpia y profesional.
- Manejo de errores.
- Funciones de AutoFill y AutoSet.

### Estructura de Funciones

- Interfaz, menú, opciones, configuración y organización general del editor.
- Iniciar un nuevo proyecto de tienda.
  - Comenzar con categorías predeterminadas o vacías.
- Gestión fácil de categorías.
  - Agregar nueva pestaña principal o pestaña de categoría.
  - Renombrar pestaña principal o pestaña de categoría.
  - Eliminar pestaña principal o pestaña de categoría.
  - Ordenar o reorganizar pestañas principales y categorías.
- Agregar paquetes al proyecto de tienda.
  - Paquete simple (1 producto dentro del paquete).
  - Paquete múltiple (hasta 6 opciones de productos dentro del paquete).
  - Paquete bundle (hasta 6 productos en una sola compra).
    - Agregar producto al paquete bundle.
    - Agregar nuevo producto.
- Guardar el proyecto de tienda.
- Abrir un proyecto guardado.
- Editar o eliminar paquetes de la tienda.
  - Editar o eliminar paquetes de producto único.
  - Editar o eliminar paquetes de múltiples productos.
  - Editar o eliminar paquetes bundle.
    - Editar o eliminar el paquete bundle.
    - Editar o eliminar un producto.
- Vista previa de tus DataTables.
- Exportar tu proyecto de tienda.

### Novedades

- Se agregó soporte para servidores basados en MueMU, Louis Emulator, Matrix Emulator y proyectos similares.
- Se agregó soporte nativo para ítems Socket.
